# **App Name**: XPS Creative Landing Page

## Core Features:

- Hero Section: Visually stunning hero section with key message and image integration.
- About Section: Showcase mission and capabilities, backed with concise descriptions
- Contact Section: Implement 'Marca um papo' (schedule an appointment) link.
- Footer Section: Include the copyright message.

## Style Guidelines:

- Background color: Pale off-white, almost a very light grey (#F5F5F5) to offer contrast with the image but remain clean.
- Primary color: Deep brown (#4A3007) to match brand image.
- Accent color: Light beige (#F5EBD7) to contrast with the background and primary colors
- Body and headline font: 'Inter' sans-serif. Note: currently only Google Fonts are supported.
- Responsive layout adapting to various screen sizes, in a single-page format.
- Subtle fade-in effects on page load and section transitions using Tween CSS.
